🚀 FetSync - Smart FET Timetable Manager

FetSync is a modern, fast, and secure web-based SaaS application designed to refine, filter, and beautifully print school timetables generated by the FET software.

While the User Interface is tailored in Arabic for the North African dual-shift educational system (Morning/Afternoon), the underlying architecture is a robust, Vanilla JavaScript XML parser that can be adapted to any FET timetable structure globally.

✨ Key Features

🔒 100% Client-Side Architecture: Total privacy. All XML parsing and data manipulation is done directly within the user's browser using the native DOMParser API. No server uploads, no backend, zero data latency.

🛠️ Smart XML Refiner (correc.html):

Intelligent Deduplication: Automatically strips out repetitive sub-activities (e.g., tutorials, practicals) and keeps only the core subjects to declutter the timetable.

Automated Blacklisting: Safely removes complex, non-essential subjects (like Physical Education, Art, Music) that typically break FET generation algorithms.

Garbage Collection: Includes a dynamic garbage collector that scans for orphaned Space/Time constraints resulting from deleted activities and purges them to prevent XML corruption.

Structural Normalization: Forces strict daily limits (4 days, 4 hours per shift) and manipulates Teacher/Student min-hour constraints dynamically.

🖨️ Custom Print Engine (print.html): * A dedicated rendering engine built with Tailwind CSS.

Converts complex XML matrices into clean, printable HTML tables for both Teachers and Classes.

Utilizes advanced @media print CSS rules for perfect A4 pagination.

📂 Project Architecture

FetSync is built with zero heavy dependencies to ensure lightning-fast execution:

HTML5 & Vanilla JS: Core logic and XML manipulation.

Tailwind CSS (CDN): Modern, responsive, and beautiful UI styling.

File Structure

index.html - The SaaS Landing Page with modern hover physics and ambient UI.

correc.html - The core Data Refiner and XML manipulation engine.

print.html - The reverse-engineering parser and print layout generator.
